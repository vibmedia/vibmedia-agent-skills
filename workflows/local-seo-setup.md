# /local-seo-setup — Local SEO Setup

## Prerequisites
- Client brand folder set up with `profile: marketing`
- Access to client's Google My Business profile
- NAP (Name, Address, Phone) confirmed with client

## Steps

1. **GMB access verification**
   - Verify access to client's Google My Business via shared email
   - If not accessible → use `@client-coordinator` to request access
   - Check if profile is claimed and verified
   - If not claimed → guide client through verification
   - Expected: Full GMB access confirmed

2. **GMB profile audit**
   - Load `google-my-business` skill
   - Run through Profile Optimization Checklist
   - Score current completeness (Basic Info, Media, Extended Info)
   - Expected: Audit report with completeness score and gaps

3. **Optimize GMB profile**
   - Fill all missing profile fields
   - Upload photos (logo, cover, interior, exterior, team, products)
   - Set correct categories (primary + up to 9 secondary)
   - Write business description (750 chars, keywords natural)
   - Set all attributes, services, hours
   - Expected: Profile completeness 95%+

4. **NAP audit**
   - Load `local-seo` skill
   - Check NAP consistency across top 30 citation sources
   - List all inconsistencies
   - Expected: NAP audit report with consistency score

5. **Citation building**
   - Create/update listings on missing citation sources
   - Fix inconsistent NAP on existing citations
   - Prioritize: Justdial, Sulekha, Yelp, Facebook, Bing Places, Apple Maps
   - Expected: Citations created/fixed on top sources

6. **Review strategy setup**
   - Create review response templates (positive, neutral, negative)
   - Respond to any existing unresponded reviews
   - Set up review monitoring (manual or API if available)
   - Create plan for encouraging customer reviews
   - Expected: Review templates ready, existing reviews responded

7. **Google Posts schedule**
   - Create first Google Post (What's New)
   - Set weekly posting schedule
   - Prepare 4 weeks of post content
   - Expected: First post published, schedule documented

8. **Local content recommendations**
   - Identify local keyword opportunities
   - Recommend location pages for service areas
   - Suggest local blog topics
   - Expected: Local content plan documented

9. **Report & handoff**
   - Generate Local SEO Audit Report (before/after scores)
   - Document all changes made
   - Set monthly review schedule
   - Send summary to client via `@client-coordinator`
   - Expected: Report delivered, ongoing plan established
