# Envato Template Selection

> Select the best Envato Elementor Kit for any client project using a systematic filtering and evaluation process.

## When to Use

- Starting a new WordPress website project
- Client needs a specific industry template
- Evaluating multiple template options for a proposal
- Sourcing Elementor kits that match client requirements

## Before Starting

1. What industry is the client in? (e.g., plumbing, restaurant, accounting)
2. What's the budget? (Envato Elements subscription or one-time purchase?)
3. How many pages does the site need?
4. Does the site need dynamic content (CPTs, repeaters)?
5. Are there specific design preferences? (modern, minimal, corporate, etc.)

## Core Framework

### Search & Filter Protocol

```
1. Go to Envato Elements (elements.envato.com)
2. Navigate to WordPress section
3. Search for industry keyword (e.g., "plumbing")
4. Apply filters:
   ├── Category: WordPress
   ├── Sub-category: Elementor Kits (CRITICAL — not themes)
   ├── Sort: Most relevant or Most popular
   └── Refine: Compatible with Elementor
5. Browse results (first 2-3 pages)
6. Shortlist 3-5 candidates
7. Evaluate using criteria below
```

### Template Evaluation Criteria

| Criteria | Weight | How to Check |
|----------|--------|-------------|
| **Industry fit** | 30% | Does the layout match the business type? |
| **Design quality** | 25% | Modern, clean, professional? No dated patterns? |
| **Page completeness** | 15% | Home, About, Services, Contact included? Other needed pages? |
| **Responsiveness** | 15% | Preview mobile and tablet views |
| **Plugin compatibility** | 10% | What plugins are required? Do we have them in Google Drive? |
| **Dynamic readiness** | 5% | Can it support CPTs/JetEngine if needed? |

### Plugin Dependency Management

> 🔴 **ALWAYS check Google Drive plugin repository before purchasing.**

| Step | Action |
|------|--------|
| 1 | List all plugins required by the template |
| 2 | Check Google Drive repository for pro/paid plugins |
| 3 | Mark which are available vs. need sourcing |
| 4 | For free plugins: install from WordPress.org |
| 5 | For paid plugins in Drive: upload manually |
| 6 | For missing paid plugins: Flag to human, do NOT purchase |

### Google Drive Plugin Repository

```
[Google Drive]/WordPress Plugins/
├── Elementor Pro/
├── JetEngine/
├── JetSmartFilters/
├── JetElements/
├── Other Pro Plugins/
└── README.md (version notes)
```

**Rules:**
- Always use the latest version from Drive
- After template selection, list ALL required plugins
- Cross-reference with Drive inventory before proceeding
- If a plugin is missing → notify human, don't buy

### Common Elementor Kit Contents

| Component | Typical Inclusions |
|-----------|-------------------|
| Pages | Home, About, Services, Contact, Portfolio, Blog, 404 |
| Headers | 2-3 header variants |
| Footers | 2-3 footer variants |
| Widgets | Custom-styled Elementor widgets |
| Global styles | Typography, colors, button styles |
| Popups | Newsletter signup, cookie consent |

## Common Mistakes

- ❌ Selecting a WordPress theme instead of an Elementor Kit (different things!)
- ❌ Not checking plugin requirements before selecting template
- ❌ Buying plugins when they're already in Google Drive
- ❌ Choosing based on homepage alone — check ALL pages
- ❌ Not previewing mobile/tablet responsiveness
- ❌ Selecting a template that requires a page builder other than Elementor
